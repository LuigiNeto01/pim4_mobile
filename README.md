## 📋 Sobre

Aplicativo **Android nativo** desenvolvido em **Java** como parte do **Projeto Integrado Multidisciplinar IV (PIM IV)** da faculdade. O app complementa o sistema web do PIM IV, oferecendo uma interface móvel para os usuários.

## ✨ Funcionalidades

- 📱 Interface nativa Android
- 🔗 Integração com a API do servidor (PIM IV Backend)
- 🔐 Autenticação de usuários
- 📊 Consulta e visualização de dados

## 📦 Como executar

### Pré-requisitos
- Android Studio instalado
- JDK 11+
- Emulador Android ou dispositivo físico

```bash
git clone https://github.com/LuigiNeto01/pim4_mobile.git
cd pim4_mobile
```

1. Abra o projeto no **Android Studio**
2. Configure o arquivo `.env` com a URL da API
3. Clique em **Run** ou use `./gradlew assembleDebug`

## ⚙️ Configuração

Edite o arquivo `.env` na raiz do projeto:

```env
API_BASE_URL=http://seu-servidor:porta
```

## 🚀 Tecnologias

- **Java**
- **Android SDK**
- **Gradle (Kotlin DSL)**
- **Retrofit** (requisições HTTP)
